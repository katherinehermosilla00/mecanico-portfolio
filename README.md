# Página Web Mecánica Automotriz – Diego Gutiérrez

Sitio web desarrollado como presentación profesional para Diego Gutiérrez, estudiante de Ingeniería en Mecánica Automotriz y Autotrónica,
donde se muestran sus trabajos, experiencia y formas de contacto.

-----------------------------------------------------------------------------------------------------------------------------------------

## Descripción

Esta página tiene como objetivo presentar de forma clara los trabajos realizados en el área automotriz, incluyendo:

- Diagnóstico de vehículos  
- Mantenimiento automotriz  
- Reparación de sistemas mecánicos  
- Trabajo en autos clásicos  
- Experiencia práctica en motores  

También incluye material audiovisual que permite observar el proceso real de trabajo.

------------------------------------------------------------------------------------------------------------------------------------------

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Font Awesome  
- Google Fonts  

-------------------------------------------------------------------------------------------------------------------------------------------

## Características

- Diseño responsivo  
- Sección de perfil profesional  
- Galería de trabajos realizados  
- Videos de procesos mecánicos  
- Navegación dinámica mediante JavaScript  
- Contacto directo a través de WhatsApp y correo electrónico
